# React before and after slider

A fast, light-weight before and after image slider for React applications.

## Demo

[Here](https://sliderbna.netlify.app/)


### Installing

Through npm

```
npm i react-bna
```

## Built With

* [Webpack]
* [Babel]
* [ES6]
